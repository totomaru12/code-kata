# js-es6-kata

## environment

- node v9.11.1

## setup

```terminal
$ npm install
$ npm install -g babel-cli
(-g babel-cli needed by babel-node)
```

## test

```terminal
$ npm test
```

## run code

```terminal
$ babel-node <source file name>
```

## setup (chai)

```terminal
$ npm install chai --save-dev
```
