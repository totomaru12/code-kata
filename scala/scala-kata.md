# scala-kata

## environment

```terminal
$ scala -version
Scala code runner version 2.12.6
$ sbt --launcher-version
sbt launcher version 1.0.4
```

## setup

```terminal
$ sbt
```

## test

```terminal
$ sbt test
```

## run

```terminal
$ sbt run
```

## setup (initial)

```terminal
$ brew install scala
$ brew install sbt
$ sbt
$ mkdir -p src/main/java src/main/resources src/main/scala
$ mkdir -p src/test/java src/test/resources src/test/scala
```
