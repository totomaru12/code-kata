# typescript-kata

## environment

- macOS 10.x
- tsc -v Version 2.9.2

## setup

```terminal
$ npm install -g typescript
$ npm install
```

## test

```terminal
$ npm test
```

## setup (initial)

```terminal
$ npm init --y
$ npm --save-dev install mocha chai typescript ts-node
$ npm --save-dev install @types/mocha @types/chai
```
